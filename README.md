# 🌍 Global Mental Health Analysis (2026)

## 📌 Objective
Analyze global mental health trends across 92 countries, focusing on:
- Social media usage
- Economic inequality
- Access to mental health care

## 📊 Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Plotly Express
- Google Colab

## 🔍 Key Insights
- Positive correlation between screen time and anxiety levels
- Lower-income countries have reduced access to treatment
- Post-COVID increase in mental health disorders globally

## 📁 Dataset
Dataset from Kaggle (https://www.kaggle.com/datasets/alitaqishah/global-mental-health-crisis-index-2026)
